Cloudant is a cloud-hosted NoSQL database service (DBaaS). See online docs for more details (and demos).

Configuring the Cloudant input connector is quite simple:

  1. Select the Cloudant Account (press the button to add an account, if needed).
  2. Select the name of the target database (the DB Name field).
  3. Select the Query type to be performed on the Cloudant DB.
  4. Specify the structure of the JSON documents you will be receiving from Cloudant. If the selected Database has documents containing different structures, then you will need to specify filtering rules (step #4) based on the field contents of the required Cloudant document. In that case, just specify the structure of the document you will be processing (and ignore the others).
  5. Go to the Data Filtering tab and optionally specify filtering rules.
Tip: Make sure the JSON document field values (defined in step #3) have human readable values. This makes the filtering process (step #4) a lot easier.
