Zendesk is a very popular customer support service (helpdesk). See online docs for more details (and demos).

Configuring the Zendesk input connector is quite simple:

  1. Select the Zendesk Account (press the button to add an account, if needed).
  2. Select the Zendesk Data object that you wish to sync/migrate from the selected account.
  3. (Optional) Go to the Data Filter tab and add filtering rules, if required.
