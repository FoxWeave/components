Zendesk is a very popular customer support service (helpdesk). See online docs for more details (and demos).

Configuring the Zendesk output connector is quite simple:

  1. Select the Zendesk Account (press the button to add an account, if needed).
  2. Select the Zendesk Data object that you wish to sync/migrate to the selected account.
  3. Go to the Data Mapping tab and map the pipeline data into the Zendesk object (selected in step #2).
