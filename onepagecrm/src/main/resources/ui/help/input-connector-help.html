OnePage CRM is a Super Easy Online Sales CRM for small business. See online docs for more details (and demos).

Configuring the OnePage CRM input connector is quite simple:

  1. Select the OnePage CRM Account (press the button to add an account, if needed).
  2. Select the OnePage CRM Data object that you wish to sync/migrate from the selected account.
  3. (Optional) Go to the Data Filter tab and add filtering rules, if required.
