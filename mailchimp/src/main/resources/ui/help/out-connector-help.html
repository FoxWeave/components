MailChimp is a very popular Newsletter mailing service. See online docs for more details (and demos).

Configuring the MailChimp output connector is quite simple:

  1. Select the MailChimp Account (press the button to add an account, if needed).
  2. Once the MailChimp Account is selected, the MailChimp List drop down will be pre-filled with mailing lists from that account. Select one of these lists.
  3. Select the MailChimp Action to be performed on the selected list.
  4. Go to the Data Mapping tab and map the pipeline data for the selected MailChimp action (selected in step #3).
