Oracle is a very popular Relational Database. See online docs for more details (and demos).

Configuring the Oracle input connector is quite simple:

  1. Select the Oracle Server (press the button to add a Server, if needed).
  2. Enter the name of the Table / View from which you wish to synchronize data.
  3. Enter the Table column names.
  4. Specify the type information for each column. This should match the definition from the database itself.
  5. Go to the Data Filter tab and configure the filtering rules.
