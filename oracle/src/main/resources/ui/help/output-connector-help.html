Oracle is a very popular Relational Database. See online docs for more details (and demos).

Configuring the Oracle output connector is quite simple:

  1. Select the Oracle Server (press the button to add an account, if needed).
  2. Enter the name of the Table to which you wish to synchronize data.
  3. Enter the Table column names, as well as the type information for each column. This should match the definition from the database itself.
  4. Go to the Add / Update tab and configure how you would like the data to be added to the table.
  5. Go to the Data Mapping tab and configure the data mappings.
