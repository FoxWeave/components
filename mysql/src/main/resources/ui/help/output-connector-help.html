MySQL is a very popular open source Relational Database. See online docs for more details (and demos).

Configuring the MySQL output connector is quite simple:

  1. Select the MySQL Server (press the button to add an account, if needed).
  2. Enter the name of the Database to which you wish to synchronize data.
  3. Enter the insert/update SQL Statement. Auto-complete of the statement value field mappings is supported. See Tip below.
Tip: When creating a MySQL Server (after pressing the button), do NOT enter a local loopback address ("localhost" or "127.0.0.1") as the MySQL Server Host name/IP. Remember that this MySQL Server Host address needs to be resolvable on the FoxWeave target on which you chose to start/run the integration i.e. on an On-Premise or On-PaaS host machine.
Tip: When entering the values field tokens in the SQL Statement (step #3), use the 'Ctrl+Enter' keyboard shortcut to auto-complete the statement value field mappings i.e. the '#<in-field-name>' tokens as seen in the following example:

insert into subscriptions (id, firstn, lastn, eaddress) values (#id, #fname, #lname, #email)

Note that the cursor needs to be active in the SQL Statement textarea. You can also press the right arrow button to activate the auto-completions.
